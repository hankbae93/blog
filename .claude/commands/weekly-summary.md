# /weekly-summary - 주간 트렌드 요약

이번 주의 데일리 인사이트를 종합하여 **주간 요약 리포트**를 생성합니다.

## 입력

- `generated/insights/YYYY-MM-DD.md` (이번 주 모든 파일)
- `generated/trends/history.json` (트렌드 히스토리)
- `generated/sources/YYYY-MM-DD.json` (이번 주 모든 소스)

## 실행 조건

매주 일요일에 실행하며, 지난 7일간의 데이터를 종합합니다.

## 출력 구조

```markdown

> **요약**: [이번 주를 관통하는 핵심 메시지]
```

## 파일 저장

- **경로**: `generated/summaries/weekly/{YYYY}-W{WW}.md`
- **인코딩**: UTF-8

## 주차 계산

ISO 8601 기준으로 주차를 계산합니다.
- 예: 2026-02-01이 속한 주 = 2026-W05

## 분석 지침

1. **정량 데이터 우선**: 숫자로 표현할 수 있는 것은 숫자로
2. **트렌드 변화 강조**: 지난주 대비 상승/하락 표시
3. **액션 지향**: 모든 인사이트는 "그래서 뭘 해야 하나"로 연결
4. **중복 제거**: 같은 제품/트렌드가 여러 번 등장하면 통합
