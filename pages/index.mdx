# 1인 개발자 시대, 가장 어려운 건 "무엇을 만들 것인가"

최근 1인 개발자, 인디해커가 정말 많아졌습니다.
AI 덕분에 개발 허들은 낮아졌지만, 오히려 "뭘 만들지"를 정하는 게 더 어려워진 것 같습니다.

저도 같은 고민을 하는 1인 개발자입니다.
그래서 직접 트렌드를 분석하고, 아이디어 힌트를 정리하는 블로그를 시작했습니다.

📌 매일 아침 10시 업데이트
📌 Product Hunt, Hacker News, GitHub Trending 등 9개 소스 분석
📌 두 가지 관점으로 평가
   - Profit Track: 빠른 수익화 가능성
   - Essence Track: 본질적 문제 해결
📌 상위 아이디어는 PRD(Product Requirements Document)까지 작성

혼자 고민하지 말고, 트렌드에서 힌트를 얻어보세요.
저도 여기서 다음에 만들 것을 찾고 있습니다.
