### 카드 3: AgentMeter — AI 코딩 에이전트 비용 추적기

**조합 공식**: Dev.to "Claude Code 비용을 모른다" + PH Mngr(100대 에이전트 병렬 실행) + Cosyra(모바일 에이전트 관리) = 에이전트 비용/성과 통합 대시보드

**문제 발견 경로**:
- Dev.to: "You're spending money on Claude Code and have no idea how much" — 비용 추적 페인포인트 직접 언급
- Product Hunt: Mngr(수백 개 에이전트 병렬), Cosyra(모바일 에이전트) — 에이전트 수가 폭증하면 비용 관리가 핵심

**문제**: AI 코딩 에이전트(Claude Code, Cursor, Copilot)를 여러 개 사용하는 개발자가 총 비용을 파악하기 어렵고, 어떤 에이전트가 실제로 생산성을 높이는지 모른다

**솔루션**: AI 코딩 도구별 API 사용량, 비용, 생성된 코드 품질을 한 곳에서 추적하는 가벼운 CLI + 웹 대시보드
