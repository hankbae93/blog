## 1. 주간 트렌드 TOP 5

### 1. AI 에이전트 인프라 — "에이전트를 위한 도구" 시장 형성 🔥 Rising
**이번 주 등장 횟수**: 4회 (매일)
**관련 제품/기사**:
- [AgentPulse](https://www.producthunt.com/products/rectify) — OpenClaw 시각적 관리 대시보드 (PH 225표)
- [Offsite](https://www.producthunt.com/products/offsite-2) — 인간+에이전트 통합 조직도 (PH 444표)
- [Grass](https://www.producthunt.com/products/grass-claude-code-from-your-phone) — 코딩 에이전트 전용 VM (PH 258표)
- [Freestyle](https://www.freestyle.sh/) — 코딩 에이전트 샌드박스 (HN 206점)

**왜 주목해야 하나**:
2025년엔 "어떤 에이전트를 쓸 것인가"가 질문이었다면, 2026년엔 "여러 에이전트를 어떻게 관리할 것인가"로 전환. 관리(AgentPulse/Offsite), 실행환경(Grass), 안전(Freestyle), 워크플로우(n8n)가 같은 주에 동시에 주목받은 것은 시장 형성 초기의 강한 신호다.

**1인 개발자 기회**:
에이전트 실행 로그 대시보드, MCP 기반 에이전트 스킬 마켓플레이스, 에이전트 비용·성과 추적 도구. 에이전트 "자체"보다 "주변 인프라"가 더 현실적인 비즈니스 기회.

### 2. 바이브코딩 품질 위기 — 작동하지만 부채를 쌓는 코드 🔥 Rising
**이번 주 등장 횟수**: 4회 (매일)
**관련 제품/기사**:
- [The peril of laziness lost](https://bcantrill.dtrace.org/2026/04/12/the-peril-of-laziness-lost/) — Bryan Cantrill의 AI 코드 비효율 경고 (HN 273점)
- [ML promises to be profoundly weird](https://aphyr.com/posts/411-the-future-of-everything-is-lies-i-guess) — Aphyr의 바이브코딩 비판 (HN 352점 + Lobsters 159점)
- [수백만 JS 개발자 RAT 감염](https://www.youtube.com/watch?v=o7NYXvYohYk) — 공급망 보안 위협 (Fireship 67만뷰)
- [Nicelydone MCP](https://www.producthunt.com/products/nicely-done) — AI 생성 UI 획일화 문제 해결 (PH 120표)

**왜 주목해야 하나**:
AI가 "작동하는 코드"를 빠르게 만드는 데는 성공했지만, 효율성(Cantrill), 신뢰성(Aphyr), 보안(공급망 공격), 디자인(획일적 UI)에서 심각한 기술 부채를 만들고 있다. 찬반 양극화가 극심하며, 이 갈등 자체가 시장 기회.

**1인 개발자 기회**:
AI 코드 품질 스코어 카드 (GitHub Action), 바이브코딩 가드레일 CLI, 의존성 신뢰도 대시보드. 바이브코딩 찬성파는 "안심하고 쓸 수 있어서", 반대파는 "최소한의 안전망"으로 양쪽 모두 수요.

### 3. "생성 AI"에서 "변환 AI"로 — 기존 데이터 재가공이 승리 🔥 Rising
**이번 주 등장 횟수**: 3회
**관련 제품/기사**:
- [Brila](https://www.producthunt.com/products/brila-2) — 구글 맵스 리뷰 → 원페이지 사이트 (PH 1,021표, 이번 주 최고)
- [ProdShort](https://www.producthunt.com/products/prodshort) — 회의 발화 → 숏폼/SNS 포스트 (PH 528표)
- [Velo](https://www.producthunt.com/products/velo-4) — 거친 화면녹화 → AI 편집 비디오 (PH 466표)

**왜 주목해야 하나**:
"콘텐츠 생성" AI는 할루시네이션과 "AI스럽다"는 거부감을 받지만, "콘텐츠 변환" AI는 원본이 존재하므로 신뢰도가 높다. Brila의 1,021표는 이 패턴의 강력한 검증. ProdShort의 "생성하지 않고 캡처한다"는 포지셔닝이 핵심.

**1인 개발자 기회**:
네이버/카카오맵 리뷰 → 소상공인 랜딩페이지 (Brila 한국 버전), GitHub 커밋 로그 → 주간 개발 리포트, 앱 리뷰 불만 → 경쟁 기회 분석 도구.

### 4. AI 비용 최적화 — "AI를 쓰는 것"에서 "AI를 싸게 쓰는 것"으로 📈 Emerging
**이번 주 등장 횟수**: 3회
**관련 제품/기사**:
- [Edgee Codex Compressor](https://www.producthunt.com/products/edgee) — 입력 토큰 49.5% 절감, 비용 35.6% 감소 (PH 131표)
- [ChatGPT Pro $100/월 출시](https://techcrunch.com/2026/04/09/chatgpt-pro-plan-100-month-codex/) — AI 프리미엄 가격대 형성
- ["Claude Code 무료로 쓰는 법"](https://www.youtube.com/watch?v=AjoD5GqosTk) — YouTube 25만뷰, 비용 민감도 확인

**왜 주목해야 하나**:
AI 도구 비용이 팀/회사 단위로 무시할 수 없는 수준이 되면서 "비용 최적화"가 독립 제품 카테고리로 형성. ChatGPT Pro $100/월, Claude Max $200/월 시대에 비용 관리는 필수.

**1인 개발자 기회**:
AI API 사용량 모니터링/최적화 대시보드, 멀티 AI 도구 비용 비교 트래커. Edgee처럼 "게이트웨이" 형태의 비용 절감 도구.
