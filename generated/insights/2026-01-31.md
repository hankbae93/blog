## Product Hunt 오늘의 제품

### 1. Leapility ⬆️ 393
![thumbnail](https://ph-files.imgix.net/d7e28bd1-51bd-41a7-9ad9-8e16268c2164.png?auto=format)

#### 이 제품은 무엇인가?
반복적인 업무 워크플로우를 AI가 학습해서 "플레이북"으로 만들어주는 자동화 도구다. 기존의 Zapier나 Make처럼 사용자가 일일이 트리거와 액션을 설정하는 게 아니라, AI가 사용자의 작업 패턴을 관찰하고 스스로 워크플로우를 구성한다.

예를 들어, "매주 월요일 Slack에서 팀 주간 보고를 요청하고, 응답을 취합해서 Notion에 정리하는" 작업을 몇 번 수동으로 하면, AI가 이를 학습해서 다음부터는 자동으로 처리한다.

#### 왜 주목할만한가?
1인 개발자의 가장 큰 병목은 "반복 업무에 시간을 빼앗기는 것"이다. 이 제품은 자동화 설정 자체를 AI에게 맡긴다는 점에서 한 단계 더 진화했다. 설정 시간 = 0에 가까워진다.

| 구분 | 내용 |
|------|------|
| 타겟 사용자 | 반복 업무가 많은 소규모 팀, 1인 사업가 |
| 기존 대비 차별점 | 사용자가 설정하는 게 아니라 AI가 패턴을 학습해서 자동화 |
| 수익 모델 | SaaS 구독 (프리미엄 예상) |
| 제작자 | 6명 팀 |

**배울 점**: "자동화 도구"를 만들 때, 설정 과정 자체를 없애는 방향으로 사고하라. 사용자는 설정하고 싶지 않다.

### 2. Kalendar.work ⬆️ 267
![thumbnail](https://ph-files.imgix.net/fe91c2fb-fcd5-4f50-8ccd-99c479f3376d.gif?auto=format)

#### 이 제품은 무엇인가?
Calendly의 완전 무료 대안이다. 미팅 예약 링크 생성, 캘린더 연동, 타임존 자동 변환 등 Calendly의 핵심 기능을 모두 갖추고 있으면서 무료로 제공한다.

#### 왜 주목할만한가?
"유료 서비스의 무료 클론"은 항상 관심을 끈다. 특히 Calendly처럼 1인 개발자가 자주 쓰는 도구라면 더더욱. 2인 팀이 만들었다는 점에서 1인 개발자도 충분히 비슷한 제품을 만들 수 있다는 증거다.

| 구분 | 내용 |
|------|------|
| 타겟 사용자 | Calendly 요금이 부담되는 프리랜서, 소규모 팀 |
| 기존 대비 차별점 | 핵심 기능 전부 무료 |
| 수익 모델 | 무료 (광고? 프리미엄 업셀?) |
| 제작자 | 2명 팀 |

**배울 점**: 검증된 시장의 유료 제품을 무료로 클론하는 전략은 여전히 유효하다. 다만 지속가능한 수익 모델이 관건.

| 기존 대비 차별점 | 단순 플레이리스트가 아닌 "그 시대 방송 경험" 재현 |

### 4. Sheetsbase ⬆️ 167
![thumbnail](https://ph-files.imgix.net/a5b78d22-3c58-4bf8-83ea-a24b922620b4.png?auto=format)

#### 이 제품은 무엇인가?
구글 시트에서 복잡한 수식을 만들어야 할 때, 자연어로 설명하면 AI가 수식을 생성해주는 크롬 확장 프로그램이다. "매출에서 비용을 빼고 세금 10%를 적용한 값"이라고 입력하면 해당 수식을 자동 생성한다.

#### 왜 주목할만한가?
"구글에서 엑셀 수식 검색하는 시간"을 없애준다. 솔로 메이커가 만든 크롬 확장으로, 1인 개발자가 만들기에 적합한 규모의 제품이다.

| 구분 | 내용 |
|------|------|
| 타겟 사용자 | 구글 시트를 자주 쓰지만 수식에 익숙하지 않은 사용자 |
| 기존 대비 차별점 | 자연어 → 수식 변환 + 단축키 지원 |
| 수익 모델 | 프리미엄 |
| 제작자 | 1명 (솔로) |

**배울 점**: 크롬 확장은 1인 개발자에게 최적의 제품 형태. 배포가 쉽고, 특정 도구 사용자를 정확히 타겟팅할 수 있다.

### 5. Superagent from Airtable ⬆️ 133
![thumbnail](https://ph-files.imgix.net/21e5f709-138d-481b-931c-ad23d9079f2f.svg?auto=format)

#### 이 제품은 무엇인가?
Airtable에서 출시한 멀티 에이전트 시스템이다. 여러 AI 에이전트가 협업해서 "완성된 작업물"을 만들어낸다. 예를 들어, "경쟁사 분석 보고서를 만들어줘"라고 하면 데이터 수집 에이전트, 분석 에이전트, 문서 작성 에이전트가 각자 역할을 수행해서 최종 보고서를 완성한다.

#### 왜 주목할만한가?
Airtable 같은 대형 서비스가 멀티 에이전트를 정식 기능으로 출시했다는 건, 이 기술이 실험 단계를 벗어났다는 신호다. 1인 개발자도 비슷한 멀티 에이전트 워크플로우를 자신의 제품에 도입할 때가 됐다.

| 구분 | 내용 |
|------|------|
| 타겟 사용자 | Airtable을 업무 자동화에 쓰는 팀 |
| 기존 대비 차별점 | 단일 AI가 아닌 여러 에이전트의 협업 |
| 수익 모델 | Airtable 구독 내 기능 |
| 제작자 | Airtable (대기업) |

**배울 점**: 멀티 에이전트 아키텍처가 프로덕션 레디가 됐다. LangGraph, CrewAI 같은 프레임워크를 활용해 자신의 제품에도 적용해볼 때.
