### 1. AI 에이전트 수직 확장 — 코딩에서 디자인·CRM·모바일까지 🔥 Rising
**이번 주 등장 횟수**: 매일 (6/6일)
**관련 제품/기사**:
- [MuleRun](https://www.producthunt.com/products/mulerun) (436↑) — 자기진화형 AI 에이전트
- [Manus AI Desktop](https://www.producthunt.com/products/manus) (379↑) — AI의 데스크톱 진입, 로컬 파일/앱 직접 조작
- [Claude Dispatch](https://www.producthunt.com/products/claude) (473↑) — 폰에서 문자로 데스크톱 Claude 원격 제어
- [Design Agent by Lokuma](https://www.producthunt.com/products/lokuma-ai) (370↑) — 에이전트가 호출하는 "디자인 지능 레이어"
- [Claude Cowork Projects](https://www.producthunt.com/products/claude-cowork-projects) (353↑) — 프로젝트별 AI 워크스페이스
- [Lightfield AI CRM](https://www.producthunt.com/products/lightfield) (406↑) — 이메일/미팅에서 CRM 자동 구축
- [GitAgent](https://www.producthunt.com/products/gitagent-2) (211↑) — 에이전트 설정을 Git으로 버전관리
- Stanford "코드 작성에서 에이전트 관리로" (22만 뷰)

**왜 주목해야 하나**:
지난주(W11)까지 에이전트의 "인프라 레이어"(결제·호스팅·보안)가 형성되었다면, 이번 주는 에이전트의 **활용 범위**가 폭발적으로 확장되었다. 데스크톱(Manus) → 모바일(Dispatch) → 디자인(Design Agent) → 영업(Lightfield CRM) → 프로젝트 관리(Cowork)까지, PH 상위 제품의 80%가 에이전트 관련이다. Design Agent의 "에이전트를 위한 도구"라는 B2A(Business-to-Agent) 모델은 완전히 새로운 카테고리를 열었다.

**1인 개발자 기회**:
에이전트 비용 모니터링 대시보드 — 여러 에이전트(Claude, Codex, Gemini)를 동시에 운영하면 API 비용 추적이 필수가 됨. 월 $9.99 SaaS로 실시간 비용·토큰 사용량 대시보드 + 최적화 제안. Design Agent처럼 "에이전트 체인에 끼워넣는 미들웨어" 형태도 유망.

### 2. 바이브코딩 → 바이브 디자인: AI가 UI까지 삼키다 🔥 Rising
**이번 주 등장 횟수**: 6회
**관련 제품/기사**:
- [Stitch 2.0 by Google](https://www.producthunt.com/products/stitch-2-0-by-google-2) (534↑) — "바이브 디자인"이라는 신조어를 만든 AI 네이티브 디자인 도구
- [Netlify.new](https://www.producthunt.com/products/netlify) (239↑) — 프롬프트 → 즉시 프로덕션 URL
- [Composer 2 by Cursor](https://www.producthunt.com/products/cursor) (278↑) — 자체 프론티어급 코딩 모델 ($0.50/M)
- "Vibe Coder vs Programmer" (YouTube ~2.9M views)
- "전 구글맵 PM이 Palantir를 바이브코딩" (YouTube ~530K views)
- Lobsters "EnshittifAIcation" (116점) — 반대 담론도 동시 성장

**왜 주목해야 하나**:
Google이 직접 "바이브 디자인"이라는 용어를 사용한 것은 이 트렌드를 공식화한 선언이다. DESIGN.md 패턴(코드의 CLAUDE.md처럼 디자인에서도 AI 협업 프로토콜)의 도입은 AI-인간 협업의 새 표준을 제시한다. 동시에 Netlify.new의 "프롬프트 → 라이브 URL" 파이프라인은 코드 작성 자체를 건너뛸 수 있게 만들었다. 바이브코딩의 경계가 코딩을 넘어 디자인·배포까지 확장된 주.

**1인 개발자 기회**:
산업 특화 UI 프롬프트 라이브러리 — Stitch 2.0의 DESIGN.md 패턴을 활용, 특정 산업(부동산, 요식업, 클리닉)에 맞춘 UI 프롬프트 템플릿 모음을 Gumroad에서 $29에 판매. 또는 바이브코딩 결과물의 접근성·보안·성능을 자동 검증하는 린터 CLI.
