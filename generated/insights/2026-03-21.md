**문제**: AI 코딩 도구(Cursor, Claude Code, OpenCode)로 코드를 생성하지만, 생성된 코드에 보안 취약점, 성능 이슈, 안티패턴이 숨어 있는지 확인하기 어렵다. 특히 1인 개발자는 코드 리뷰어가 없다.

**솔루션**: AI가 생성한 코드의 diff를 자동으로 분석해서 보안 취약점, 성능 이슈, 코드 스멜을 잡아주는 CLI 도구. git commit 훅으로 자동 실행.

**MVP 스펙 (핵심 기능 3개)**:
1. Diff 분석 — git diff를 파싱해서 AI 생성 코드 변경분 추출
2. 보안 스캔 — 하드코딩된 시크릿, SQL 인젝션, XSS 패턴 탐지
3. 품질 리포트 — 이슈별 심각도 + 수정 제안을 터미널에 출력

**기술 스택**: Node.js CLI + Claude API (코드 분석) + git hooks
