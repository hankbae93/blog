## 1. 주간 트렌드 TOP 5

### 1. AI 에이전트 풀스택 생태계 🔥 Rising
**이번 주 등장 횟수**: 매일 (7/7일)
**관련 제품/기사**:
- [Prava](https://www.producthunt.com) (267↑) - 에이전트 결제 인프라
- [Agent 37](https://www.producthunt.com) (333↑) - $3.99/월 에이전트 호스팅
- [ClawSecure](https://www.producthunt.com) (271↑) - 에이전트 보안 레이어
- [Cal.com Agents](https://www.producthunt.com) (203↑) - 에이전트 스케줄링
- [Google Workspace CLI](https://www.producthunt.com) (319↑) - 에이전트용 CLI 인터페이스
- [Naoma AI Demo Agent](https://www.producthunt.com) (529↑) - AI 세일즈 데모 에이전트

**왜 주목해야 하나**:
지난주(W10)에는 "에이전트 파이프라인 완성"이 키워드였다면, 이번 주는 결제(Prava) → 호스팅(Agent 37, $3.99) → 보안(ClawSecure) → 스케줄링(Cal.com) → CLI(Google Workspace)까지 에이전트가 실제 비즈니스에 배포되기 위한 모든 인프라 레이어가 동시에 등장했다. 에이전트 보안 투자만 $190M 규모로 추산.

**1인 개발자 기회**:
에이전트 모니터링/옵저버빌리티 SaaS — 에이전트가 실제 결제·작업을 수행하면 "무엇을 했는지" 추적하는 대시보드 수요 급증. Agent 37의 $3.99 호스팅 위에 월 $9.99 모니터링 애드온 모델 가능.

### 2. 바이브 코딩(Vibe Coding) 메인스트림 논쟁 🔥 Rising
**이번 주 등장 횟수**: 5회
**관련 제품/기사**:
- "Vibe Coder vs Programmer" (YouTube ~2.59M views)
- "Shall I implement it? No" (HN 839pts)
- Redox OS "LLM 없이 개발" (HN 372pts)
- Claude Code Review (PH 393↑)
- Debian AI 생성 코드 정책 논의

**왜 주목해야 하나**:
"바이브 코딩"이 개발자 밈에서 진지한 업계 논쟁으로 전환. YouTube에서 259만 뷰를 기록한 영상이 촉발점이 되어, HN에서는 "구현할까요? 아니요(839pts)"라는 반-바이브코딩 글이 화제. 동시에 Redox OS는 "LLM 없이 OS 개발"을 어필하며 372pts 획득. 코드 품질 검증 도구(Claude Code Review 393↑)도 동반 성장.

**1인 개발자 기회**:
AI 생성 코드 품질 감사 도구 — "바이브 코딩으로 만든 코드베이스"를 분석해 보안 취약점·기술부채를 리포트하는 CLI 도구. Amazon GenAI 장애 사례가 수요 증거.

### 3. 로컬 AI / 프라이버시 운동 📈 Accelerating
**이번 주 등장 횟수**: 4회
**관련 제품/기사**:
- "Can I run AI locally?" (HN 853pts)
- Montana Right to Compute Act (HN 234pts)
- GStack - 로컬 AI 스택 (PH 255↑)
- DynamicLake - 프라이버시 우선 데이터 레이크 (PH 366↑)

**왜 주목해야 하나**:
"로컬에서 AI 돌릴 수 있나요?"가 HN 853pts로 이번 주 기술 토론 1위. Montana주는 "컴퓨팅 권리법"을 발의하며 프라이버시·자기주권 컴퓨팅이 입법 단계까지 진입. 제품 측면에서도 GStack(로컬 AI 스택)과 DynamicLake(프라이버시 우선)이 동시 출시.

**1인 개발자 기회**:
로컬 LLM 원클릭 설치 + 벤치마크 도구 — "내 맥북에서 어떤 모델이 가장 잘 돌아가는지" 자동 테스트하는 앱. 하드웨어별 최적 모델 추천까지.

### 4. AI 코드 품질·검증 위기 ⚠️ Warning
**이번 주 등장 횟수**: 4회
**관련 제품/기사**:
- Amazon GenAI 코드로 인한 서비스 장애 (HN)
- Claude Code Review (PH 393↑)
- Debian AI 생성 코드 정책 논의
- Glassworm 유니코드 공격 (HN 206pts)

**왜 주목해야 하나**:
Amazon의 AI 생성 코드 장애가 실제 대규모 서비스에서 발생하며 "AI 코드를 그냥 쓰면 안 된다"는 인식 확산. Debian은 AI 생성 코드에 대한 공식 정책을 논의 중. Claude Code Review(393↑)가 이 틈새를 공략. Glassworm 유니코드 공격(206pts)은 AI가 감지하기 어려운 보안 취약점의 새로운 벡터를 보여줌.

**1인 개발자 기회**:
PR별 "AI 생성 코드 비율" 감지 + 리스크 스코어링 GitHub Action — 팀에서 AI 코드 품질 관리를 자동화하려는 수요. Freemium 모델로 월 $19 Pro.

### 5. 콘텐츠 로컬라이제이션 파이프라인 📈 Accelerating
**이번 주 등장 횟수**: 3회
**관련 제품/기사**:
- [Visual Translate](https://www.producthunt.com) (492↑) - 이미지 내 텍스트 번역
- [Fish Audio S2](https://www.producthunt.com) (276↑) - 음성 합성/복제
- FFmpeg-over-IP (HN) - 원격 미디어 처리

**왜 주목해야 하나**:
이미지 번역(Visual Translate 492↑) + 음성 복제(Fish Audio) + 미디어 처리(FFmpeg-over-IP)가 동시에 등장하며, "콘텐츠를 현지화하는 전체 파이프라인"이 AI로 자동화되는 흐름. 지난주 대비 새로운 트렌드.

**1인 개발자 기회**:
유튜브 영상 자동 현지화 SaaS — 영상 업로드 → 자막 추출 → 번역 → 음성 더빙 → 썸네일 번역까지 원스톱. 크리에이터 시장 타겟.
