### 1. Hugo ⬆️ 416 | 💬 144 | ⭐ 4.75
![thumbnail](https://ph-files.imgix.net/dcd6e498-8d0c-41ee-8265-642dbb43862a.gif?auto=format)

🔗 [Product Hunt](https://www.producthunt.com/products/crisp) | [공식 사이트](https://crisp.chat)
📂 Customer Communication, SaaS, Artificial Intelligence

#### 이 제품은 무엇인가?
Hugo는 반복적인 고객 지원 문의를 자동으로 해결하는 AI 에이전트다. 기존 챗봇과 달리 대화를 완전히 해결하고 실제 업무까지 자동 수행한다. Crisp 팀이 만들었으며, "티켓당 1달러" 과금 모델 대신 합리적인 가격을 제시한다.

- 핵심 기능: 고객 문의 자동 해결, 업무 자동화, 안전한 end-to-end 처리
- 작동 방식: 기존 고객지원 시스템과 연동하여 AI가 대화를 이해하고 해결

#### 왜 주목할만한가?
"티켓당 1달러" 과금 모델을 비판하며 등장한 점이 인상적이다. Intercom의 Fin이 티켓당 과금으로 비용 부담이 크다는 불만이 많았는데, Hugo는 이 페인포인트를 정면으로 공략했다.

| 구분 | 내용 |
|------|------|
| 타겟 사용자 | SaaS 운영자, 고객지원팀이 있는 스타트업 |
| 기존 대비 차별점 | 티켓당 과금 아닌 합리적 가격, Crisp 팀의 검증된 기술력 |
| 수익 모델 | 월 구독 (티켓당 과금 아님) |
| 제작자 | Crisp 팀 (4명) |
| 커뮤니티 반응 | 댓글 144개, 평점 4.75/5.0 |

**배울 점**: 경쟁사의 가격 정책 불만을 정면으로 공략한 포지셔닝. "1달러/티켓이 부담되시죠?"라는 명확한 메시지.

### 2. Atoms ⬆️ 311 | 💬 56 | ⭐ -
![thumbnail](https://ph-files.imgix.net/5a87efce-da3a-41a5-b945-d6a36eaa863d.webp?auto=format)

🔗 [Product Hunt](https://www.producthunt.com/products/atoms-5)
📂 Artificial Intelligence, Development, Vibe coding

#### 이 제품은 무엇인가?
Atoms는 "바이브 비즈니스 팀"을 표방하는 AI 제품이다. 아이디어만 주면 시장 조사, 제품 설계, 프론트엔드/백엔드 개발, 인증/결제 연동까지 해서 실제로 돈을 받을 수 있는 앱을 배포해준다. 프로토타입이 아닌 실제 제품을 만들어준다고 강조한다.

- 핵심 기능: 시장조사 → 설계 → 개발 → 배포까지 전 과정 자동화
- 작동 방식: 아이디어 입력 → AI가 전체 비즈니스 플로우 실행

#### 왜 주목할만한가?
"Vibe coding"이 토픽에 포함된 점이 시사적이다. 코딩의 진입장벽이 완전히 사라지는 시대를 예고한다. 1인 개발자에게는 경쟁자이자 도구가 될 수 있다.

| 구분 | 내용 |
|------|------|
| 타겟 사용자 | 비개발자 창업자, 아이디어만 있는 사람 |
| 기존 대비 차별점 | 프로토타입이 아닌 실제 결제 가능한 제품 배포 |
| 수익 모델 | 미공개 |
| 제작자 | 6명 팀 |
| 커뮤니티 반응 | 댓글 56개 |

**배울 점**: "Vibe coding"이라는 새로운 카테고리의 등장. 코딩 없이 제품을 만드는 시장이 빠르게 성장 중.

### 3. findable. ⬆️ 278 | 💬 33 | ⭐ 5.0
![thumbnail](https://ph-files.imgix.net/15eeaa12-94e2-4d9b-b424-13c87736b2f2.png?auto=format)

🔗 [Product Hunt](https://www.producthunt.com/products/findable-3)
📂 Marketing, SEO, Search

#### 이 제품은 무엇인가?
findable은 AEO(Answer Engine Optimization) 플랫폼이다. ChatGPT, Gemini, Perplexity 같은 AI 검색 엔진에서 내 제품이 답변으로 추천되도록 최적화해주는 도구다. SEO의 다음 단계인 AEO를 무료로 시작할 수 있게 해준다.

- 핵심 기능: AI 검색 엔진 최적화, 답변 노출 분석
- 작동 방식: AI 검색 엔진들이 내 콘텐츠를 어떻게 인식하는지 분석하고 최적화 제안

#### 왜 주목할만한가?
SEO에서 AEO로의 전환을 선점한 제품이다. Google 검색 점유율이 AI 검색에 잠식당하는 상황에서 필수적인 도구가 될 수 있다.

| 구분 | 내용 |
|------|------|
| 타겟 사용자 | 마케터, 콘텐츠 크리에이터, SaaS 운영자 |
| 기존 대비 차별점 | SEO가 아닌 AEO에 특화, 무료 플랜 제공 |
| 수익 모델 | 프리미엄 (무료 플랜 포함) |
| 제작자 | 2명 팀 |
| 커뮤니티 반응 | 댓글 33개, 평점 5.0/5.0 |

**배울 점**: SEO → AEO 전환기에 새로운 니치 시장이 열리고 있다. 1인 개발자도 자신의 제품을 AI 검색에 최적화해야 한다.
