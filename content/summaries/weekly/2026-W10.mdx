## 1. 주간 트렌드 TOP 5

### 1. AI 에이전트 파이프라인의 완성 — 코딩 너머 테스트·보안·배포까지 🔥 Rising
**이번 주 등장 횟수**: 매일 (7/7일)
**관련 제품/기사**:
- [Claude Marketplace](https://www.producthunt.com/products/claude-marketplace) — PH 399표, Anthropic의 AI 솔루션 마켓플레이스
- [Anything API](https://www.producthunt.com/products/notte) — PH 380표, API 없는 웹사이트를 API로 변환
- [TestSprite 2.1](https://www.producthunt.com/products/testsprite) — PH 310표, AI 자동 테스트 생성
- [Codex Security](https://www.producthunt.com/products/openai) — PH 223표, OpenAI 보안 에이전트
- [21st Agents SDK](https://www.producthunt.com/products/21st-dev-the-npm-for-design-engineers) — PH 156표, 에이전트 배포 인프라
- [Aident AI](https://www.producthunt.com/products/aident-ai) — PH 307표, 자연어 자동화 빌더

**왜 주목해야 하나**:
이번 주의 결정적 전환은 AI 에이전트가 "코드를 생성하는 것"에서 "코드를 생성하고, 테스트하고, 보안을 점검하고, 배포하는 것"까지 확장된 것이다. TestSprite(테스트), Codex Security(보안), 21st Agents SDK(배포)가 같은 주에 PH 상위에 동시 등장한 것은 개발 파이프라인 전체가 에이전트화되고 있음을 보여준다. HN에서 "LLM은 합격 기준을 먼저 정의할 때 잘 작동한다"(210점)가 171개 댓글로 폭발한 것도 같은 맥락이다.

**1인 개발자 기회**:
에이전트 자체보다 **에이전트 간 연결·테스트·모니터링 도구**가 블루오션. MCP 브릿지, 에이전트 비용 추적 대시보드, 특정 프레임워크(Next.js, Flutter) 특화 AI 코드 검증 도구가 구체적 MVP 후보.

### 2. 바이브 코딩이 밈에서 문화로 — 정체성 논쟁 본격화 🔥 Rising
**이번 주 등장 횟수**: 매일 (7/7일)
**관련 제품/기사**:
- [Vibe Marketplace by Greta](https://www.producthunt.com/products/gretabyquestera) — PH 211표, 바이브 코딩 결과물 판매 마켓
- [Enia Code](https://www.producthunt.com/products/enia-code) — PH 230표, 능동형 AI 코드 리뷰
- YouTube "Vibe Coder vs Programmer" — Alberta Tech, 242만 뷰
- YouTube "전 Google Maps PM이 주말에 Palantir를 바이브 코딩" — Bilawal Sidhu, 35만 뷰
- HN "좋은 소프트웨어는 멈출 줄 안다" — 326점, 179댓글
- Lobsters "나는 LLM에 상담하지 않는다" — 42점, 찬반 논쟁

**왜 주목해야 하나**:
핵심 전환점은 "비개발자가 바이브 코딩을 한다"에서 "전문 개발자가 바이브 코딩을 실전에 쓴다"로 바뀌었다는 점이다. 전 Google Maps PM이 주말에 Palantir급 제품을 만든 사례(좋아요 비율 5.5%, 올해 최고)가 이를 증명한다. Vibe Marketplace가 PH 2위로 올라온 것은 "만들고 바로 파는" 바이브 이코노미가 시작되었다는 신호. 동시에 "LLM 없이 코딩하겠다"는 반론도 42점을 받으며 건재하다.

**1인 개발자 기회**:
바이브 코딩으로 만든 코드의 품질을 검증하는 "가드레일" 도구. 특히 보안 취약점·성능 병목·접근성 문제를 자동 감지하는 도구가 시급하다. 도메인 전문가(의사, 변호사, 교사)를 위한 바이브 코딩 스타터 킷도 유망.

### 3. AI 코드 검증과 보안 위기 🔥 Rising
**이번 주 등장 횟수**: 6/7일
**관련 제품/기사**:
- HN "GitHub 이슈 제목 하나가 4,000대 개발자 머신을 감염" — 305점 (Clinejection)
- HN 위키피디아 관리자 계정 대량 해킹 — 859점
- HN Knuth의 "Claude Cycles" 논문 — 485점
- HN "AI가 소프트웨어를 쓸 때 누가 검증하나?" — 151점
- HN Agent Safehouse (macOS 에이전트 샌드박싱) — 233점
- TechCrunch: Anthropic Claude가 Firefox에서 취약점 22개 발견
- PH: Golf (MCP 보안 관리 플랫폼) — 179표

**왜 주목해야 하나**:
AI 코딩 도구가 새로운 공격 벡터로 부상했다. Cline을 통한 프롬프트 인젝션이 4,000대 머신을 실제로 감염시켰고, MCP 프로토콜의 도구 설명 자체가 공격 표면이 될 수 있다는 경고가 나왔다. Knuth 같은 CS 석학이 AI 코딩에 관심을 가진다는 것(485점)은 학계-업계 경계가 허물어지는 신호이며, Anthropic이 Firefox 취약점 22개를 2주 만에 찾은 것은 AI 보안 감사가 실전 단계에 진입했음을 증명한다.

**1인 개발자 기회**:
MCP 보안 린터/스캐너(오픈소스), AI 코딩 도구 권한 관리자(샌드박스), 오픈소스 프로젝트 대상 AI 보안 감사 서비스. 에이전트 행동 로깅/감사 도구도 Agent Safehouse(233점)가 보여준 수요.

### 4. AI 모델 비용 급락과 소형 모델 실용화 🔥 Rising
**이번 주 등장 횟수**: 5/7일
**관련 제품/기사**:
- [Qwen3.5 Small](https://www.producthunt.com/products/qwen3) — PH 300표, 0.8B~9B 소형 멀티모달 모델
- [Gemini 3.1 Flash-Lite](https://www.producthunt.com/products/gemini-6) — PH 236표, $0.25/M 토큰
- HN GPT-5.3 Instant — 300점
- HN GPT-5.4 — 577점, 507 댓글
- HN Qwen 관련 글 2건 동시 화제 (505점 + 269점)
- TechCrunch: Google Play Store 수수료 30%→20% 인하

**왜 주목해야 하나**:
"더 크게"가 아닌 "더 작고 더 빠르게"가 AI 경쟁의 새 축이다. Qwen 0.8B는 스마트폰에서도 돌릴 수 있고, Gemini Flash-Lite는 $0.25/M 토큰으로 6개월 전 대비 5배 저렴하다. 여기에 Google Play 수수료 10%p 인하까지 더하면, AI 기반 앱의 마진이 크게 개선된다. 1인 개발자가 AI 기반 앱으로 수익을 내기 가장 좋은 시점이 도래했다.

**1인 개발자 기회**:
API 비용 걱정 없는 "모든 사용자 입력에 AI를 거치는" 앱(실시간 문법 교정, 상시 코드 리뷰 등). Qwen3.5를 활용한 로컬/오프라인 AI 유틸리티(번역, OCR, 요약). 프라이버시 중시 도메인(의료, 법률)에서 로컬 AI 앱.

**이번 주 등장 횟수**: 3/7일 (주 후반에 급부상)
**관련 제품/기사**:

**왜 주목해야 하나**:
Anthropic이 API 제공자에서 마켓플레이스 운영자로 진화하고 있다. Claude Marketplace는 기존 Anthropic 계약으로 바로 결제할 수 있어 구매 장벽이 극도로 낮다. 1인 개발자에게는 Claude API 기반 솔루션을 만들어 B2B로 유통할 수 있는 새로운 채널이 열린 것. "만드는 도구"와 "파는 채널"을 묶는 Greta 모델도 주목.
