iOS에서도 동시에 #34에 진입(리뷰 3,970개)했다. Android와 iOS 동시 상승은 강한 크로스 플랫폼 수요를 의미한다. "돈을 받을 수 있다"는 명확한 가치 제안이 사용자를 끌어들이고 있다.

### 카드 1: ClaimSpotter KR — 한국판 미수령 환급금 알림 앱

**조합 공식**: Payout(앱 마켓 #31, 크로스 플랫폼 동시 상승)의 "혜택 대행" 모델 + 한국 미수령 환급금 시장

**문제 발견 경로**:
- 앱 마켓: Payout이 Android #31 + iOS #34에 동시 진입, 3일 연속 상승 — 검증된 수요
- Google Trends: 한패스(외국인 금융 플랫폼) 코스닥 상장으로 금융 접근성 화제

**문제**: 한국에서 매년 수천억 원의 환급금/보상금이 미수령 상태로 방치됨. 건강보험 환급, 통신비 미환급, 세금 과납 등이 여러 기관에 흩어져 있어 개인이 일일이 확인하기 어렵다.

**솔루션**: 여러 기관의 미수령 환급금을 한 번에 조회하고, 청구까지 가이드해주는 원스톱 앱

**MVP 스펙 (핵심 기능 3개)**:
1. 미수령 환급금 통합 조회 — 건강보험, 국세청, 통신사 등 공개 API 연동
2. 원클릭 청구 가이드 — 기관별 청구 절차를 단계별로 안내
3. 알림 설정 — 새로운 환급 가능 항목 발생 시 푸시 알림

**기술 스택**: React Native + Node.js + 공공데이터 API

**수익 검증 데이터**:
- 유사 앱 [Payout]: 리뷰 285개(Android) + 3,970개(iOS), 평점 4.51~4.58, 순위 #31~#34
- Payout의 수익 모델: 보상금 일부 수수료 → 한국에서는 환급 대행 수수료 또는 프리미엄 구독
- 예상 수익 모델: 프리미엄 구독 ₩3,900/월 — 벤치마크: Payout 무료+수수료 모델

**1인 개발자 실현 가능성**: ⭐⭐⭐⭐☆ (4/5)

| 평가 항목 | 점수 | 근거 |
|-----------|------|------|
| 기술 난이도 | ⭐⭐/5 | 공공데이터 API는 대부분 REST, React Native로 크로스 플랫폼 |
| 시장 진입 장벽 | ⭐⭐/5 | 한국 시장에 직접 경쟁자가 적음 |
| 초기 유저 확보 | ⭐⭐⭐/5 | "돈 돌려받기" — 바이럴 가능성 높음, 커뮤니티/SNS 마케팅 |
| 수익화 가능성 | ⭐⭐⭐⭐/5 | Payout의 크로스 플랫폼 상승이 수요를 검증 |
| MVP 개발 기간 | 2주 | 공공 API 연동 + 기본 UI + 푸시 알림 |

**경쟁 분석**:

### 카드 2: AgentGuard — 경량 AI 에이전트 액션 로거

**조합 공식**: Cekura(PH 361표, 댓글 99개)의 에이전트 모니터링 니즈 + LiteLLM 공급망 공격(HN 484점)의 보안 우려 + Claude Computer Use(PH 1위)의 자율 행동 확대

**문제 발견 경로**:
- Product Hunt: Cekura가 댓글 99개 — AI 에이전트 모니터링 수요 폭발적
- Hacker News: LiteLLM 감염 484점 + 372댓글 — AI 도구 체인 보안 우려 최고조
- Dev.to: "AI가 코드를 쓴다. 품질은 당신이 소유한다" — 책임 소재 논의

**문제**: AI 에이전트(Claude Code, OpenClaw 등)가 자율적으로 파일 수정, API 호출, 시스템 명령을 실행하는데, 무엇을 했는지 사후에 확인하기 어렵다. Cekura 같은 엔터프라이즈 도구는 소규모 팀/1인 개발자에게 과하다.

**솔루션**: AI 에이전트의 모든 액션을 타임라인으로 기록하고, 위험 행동(파일 삭제, 외부 API 호출, 의존성 설치)을 하이라이트하는 경량 CLI 도구

**MVP 스펙 (핵심 기능 3개)**:
1. 액션 로깅 — 에이전트의 파일 변경, 명령 실행, API 호출을 자동 기록
2. 위험도 분류 — 각 액션을 안전/주의/위험으로 자동 분류
3. 타임라인 뷰 — 터미널 또는 웹 UI로 에이전트 액션 히스토리 시각화

**기술 스택**: Next.js (웹 대시보드) + Node.js CLI + SQLite (로컬 저장)

**수익 검증 데이터**:
- 유사 서비스 [Cekura]: PH 361표, 댓글 99개 — 니즈 검증
- LiteLLM 사건으로 AI 보안 도구 수요 급증 (HN 484점)
- 예상 수익 모델: 오픈소스 코어 + Pro 구독 $9/월 — 팀 기능, 클라우드 동기화

**1인 개발자 실현 가능성**: ⭐⭐⭐⭐☆ (4/5)

| 평가 항목 | 점수 | 근거 |
|-----------|------|------|
| 기술 난이도 | ⭐⭐/5 | 파일 시스템 감시 + 프로세스 로깅은 기존 라이브러리 활용 가능 |
| 시장 진입 장벽 | ⭐⭐⭐/5 | Cekura는 엔터프라이즈 대상, 개인/소규모 팀 대상 경량 도구는 공백 |
| 초기 유저 확보 | ⭐⭐⭐⭐/5 | GitHub 오픈소스 + HN/Reddit 개발자 커뮤니티 |
| 수익화 가능성 | ⭐⭐⭐/5 | 오픈소스 + Pro 모델, 에이전트 사용 증가에 비례하여 성장 |

- 이 아이디어의 차별점: 프레임워크 무관, 에이전트 불문, 경량 CLI 기반

### 카드 3: MeasureSnap — 사진 기반 실측 도구 (Photo Measures 약점 공략)

**조합 공식**: Photo Measures(iOS 유료 #39, 평점 4.57)의 "사진에 치수 기록" 니즈 + 앱 리뷰의 핵심 불만 "실제 측정을 안 해준다"

**문제 발견 경로**:
- 앱 마켓: Photo Measures가 유료 앱 #39, 리뷰 518개, $6.99
- 앱 리뷰: "이 앱은 실제로 측정을 안 한다. 그냥 사진에 선 긋는 그래픽 앱이다"(missing features 2건), "측정값이 전부 틀리다"(crash/bugs 2건)
- Product Hunt: Claude Computer Use — AI의 시각적 인식 능력 확대

**문제**: 건설 현장, 인테리어, DIY 작업자들이 현장 사진에 치수를 기록해야 하는데, Photo Measures는 "사진에 수동으로 숫자를 적는" 도구일 뿐이다. 사용자들이 원하는 것은 "사진에서 자동으로 실제 길이를 측정해주는" 도구다.

**솔루션**: 기준 물체(신용카드, A4 용지 등)를 인식하여 사진 내 물체의 실제 크기를 자동 추정하고, AR 기반으로 치수를 오버레이하는 앱

**MVP 스펙 (핵심 기능 3개)**:
1. 기준 물체 캘리브레이션 — 사진 내 알려진 크기의 물체를 지정하여 스케일 자동 계산
2. 탭 투 메저 — 사진 위의 두 점을 탭하면 실제 거리를 자동 계산
3. 치수 저장 및 공유 — 측정된 사진을 PDF/이미지로 공유

**기술 스택**: React Native + Vision API (기준 물체 인식) + Canvas (오버레이)

**수익 검증 데이터**:
- 유사 앱 [Photo Measures]: 리뷰 518개, 평점 4.57, 순위 #39, $6.99
- 앱 리뷰 불만: "실제 측정을 안 한다"(2건), "측정값이 틀리다"(2건), "선이 안 보인다"(2건)
- 예상 수익 모델: 유료 앱 $4.99 또는 프리미엄 $2.99/월 — Photo Measures의 $6.99보다 저렴하게

**1인 개발자 실현 가능성**: ⭐⭐⭐☆☆ (3/5)

| 평가 항목 | 점수 | 근거 |
|-----------|------|------|
| 기술 난이도 | ⭐⭐⭐/5 | 기준 물체 인식은 Vision API 활용 가능하나 정확도 튜닝 필요 |
| 시장 진입 장벽 | ⭐⭐/5 | Photo Measures의 리뷰 불만이 명확 — 개선점이 분명 |
| 초기 유저 확보 | ⭐⭐⭐/5 | ASO + 건설/인테리어 커뮤니티 |
| 수익화 가능성 | ⭐⭐⭐⭐/5 | $6.99 유료 앱이 #39에 있으므로 유료 모델 검증됨 |
| MVP 개발 기간 | 2주 | 기준 물체 1종(신용카드) + 탭 측정 + 저장 |

**경쟁 분석**:
- 기존 솔루션: Photo Measures($6.99), Apple 측정 앱(AR 기반), MagicPlan
- 이들의 약점: Photo Measures는 자동 측정 미지원, Apple 측정은 현장에서만 가능(사진 불가)
- 이 아이디어의 차별점: "이미 찍은 사진"에서 자동으로 실제 치수를 추정하는 기능
