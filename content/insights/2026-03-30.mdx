### 카드 3: HobbyAI Kit - 오프라인 취미 AI 가이드 프레임워크

**조합 공식**: YarnPal의 성공(100만 설치, 니치 점수 100) + Tai Chi 앱의 시니어 니치 전략 + SUN의 인터랙티브 학습 모델

**문제 발견 경로**:
- Play Store 니치: YarnPal (크로셰, 100만 설치, 니치 점수 100) - "취미 + AI" 조합 검증
- Play Store 니치: Tai Chi for Beginners (10만 설치, 니치 점수 100) - "특정 취미 + 초보자" 검증
- Product Hunt: SUN (253표, 평점 5.0) - AI 기반 인터랙티브 학습의 높은 수요

**문제**: 새로운 오프라인 취미(서예, 수채화, 도자기, 가드닝 등)를 배우고 싶은 초보자가 유튜브 영상만으로는 자신의 수준에 맞는 단계별 가이드를 찾기 어렵고, 막히면 질문할 곳이 없다.

**솔루션**: 특정 오프라인 취미에 특화된 AI 가이드 웹앱. 사진을 올리면 AI가 현재 수준을 진단하고 다음 단계를 추천하며, 질문에 답변한다.

**MVP 스펙 (핵심 기능 3개)**:
1. 수준 진단 - 사용자가 작품 사진을 올리면 AI(Claude Vision)가 현재 수준을 진단하고 개선점 제시
2. 단계별 프로젝트 - 난이도별 프로젝트 목록과 재료/도구 체크리스트 제공
3. AI Q&A - "이 부분이 잘 안 돼요" + 사진으로 질문하면 AI가 구체적 조언

**기술 스택**: Next.js + Claude Vision API + Supabase (사용자 데이터)

**수익 검증 데이터**:
