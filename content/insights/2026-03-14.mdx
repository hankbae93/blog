**조합 공식**: SleepTown의 "수면 게이미피케이션" (App Store #33) + Pinnacle의 "AI 뇌 코칭" (PH 263↑) + SleepTown 리뷰 불만 "침대 시간 유연성 부족"

**문제 발견 경로**:
- App Store: SleepTown 리뷰 1,464개, 평점 4.65, 유료 $1.99, #33
- App Store 리뷰 불만: "그룹 멤버별 다른 취침 시간 설정 불가" (3건), "기상 버튼 누르기 잊으면 기록 손실" (1건)
- PH: Pinnacle 263표, 5.0 평점 - AI 기반 생산성 코칭 수요 폭발

**문제**: SleepTown 사용자들이 "유연하지 않은 취침 시간 규칙"과 "수면 데이터만으로는 부족한 코칭"에 불만. 수면뿐 아니라 집중 시간까지 통합 관리하고 싶은 수요.

**솔루션**: 수면 + 집중 시간을 게이미피케이션으로 통합 관리하고, AI가 개인 패턴을 분석해 맞춤 코칭을 제공하는 앱

**MVP 스펙 (핵심 기능 3개)**:
1. 수면/집중 타이머 - SleepTown 스타일 게이미피케이션 (건물 짓기) + 유연한 시간 설정
2. AI 패턴 분석 - 수면/집중 데이터를 분석해 "오늘 집중력이 낮은 이유" 인사이트 제공
3. 그룹 챌린지 - 각 멤버별 다른 목표 시간 설정 가능 (SleepTown 불만 해결)

**기술 스택**: React Native + Supabase + Claude API (패턴 분석)

**수익 검증 데이터**:
- SleepTown: 리뷰 1,464개, 유료 $1.99, App Store #33
- SleepTown 리뷰 불만: "그룹 취침 시간 동일" 불만 3건 이상
- Pinnacle: PH 263표, 5.0 평점 - AI 코칭 수요 검증
- 예상 수익 모델: 일회성 구매 $2.99 + 프리미엄 AI 코칭 $4.99/월

**1인 개발자 실현 가능성**: ⭐⭐⭐☆☆ (3/5)

| 시장 진입 장벽 | ⭐⭐/5 | SleepTown 경쟁하지만 불만 포인트 명확 |

**경쟁 분석**:
- 기존 솔루션: SleepTown, Forest, Pinnacle
- 이들의 약점: SleepTown은 수면만, Forest는 집중만, Pinnacle은 게이미피케이션 없음
- 이 아이디어의 차별점: 수면+집중 통합 + 유연한 그룹 설정 + AI 코칭
