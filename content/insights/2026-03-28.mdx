### 1. Agentation ⬆️ 352 | 💬 17
![thumbnail](https://ph-files.imgix.net/12aaaba2-622c-400e-87c2-a49010f9f400.png?auto=format)

🔗 [Product Hunt](https://www.producthunt.com/products/agentation?utm_campaign=producthunt-api&utm_medium=api-v2&utm_source=Application%3A+PRD+Agent+Daily+Collector+%28ID%3A+267070%29) | [공식 사이트](https://www.producthunt.com/r/UBMKYT2Z76KGDI?utm_campaign=producthunt-api&utm_medium=api-v2&utm_source=Application%3A+PRD+Agent+Daily+Collector+%28ID%3A+267070%29)
📂 Productivity, Developer Tools, Artificial Intelligence

#### 이 제품은 무엇인가?
AI 코딩 에이전트를 위한 시각적 피드백 도구다. UI 요소를 클릭하고 메모를 추가하면, AI가 이해할 수 있는 구조화된 컨텍스트로 변환해준다. 이 결과를 Claude Code, Codex 등 AI 도구에 붙여넣으면 정확한 UI 수정이 가능하다.
- 핵심 기능: UI 어노테이션 → 구조화된 AI 컨텍스트 생성 → 클립보드에 복사
- 작동 방식: 브라우저에서 요소 클릭 + 메모 작성 → JSON 형태 컨텍스트 자동 생성

#### 왜 주목할만한가?
AI 코딩 에이전트의 가장 큰 약점은 "UI를 볼 수 없다"는 것이다. Agentation은 이 갭을 메우는 브릿지 도구로, 에이전트 생태계의 핵심 미싱 피스를 공략했다.

| 구분 | 내용 |
|------|------|
| 타겟 사용자 | AI 코딩 에이전트 사용 개발자 (Claude Code, Codex 유저) |
| 기존 대비 차별점 | 스크린샷이 아닌 "구조화된 UI 컨텍스트" — AI가 정확히 이해 가능 |
| 수익 모델 | 미공개 (프리미엄 예상) |
| 커뮤니티 반응 | 댓글 17개 — "에이전트 워크플로우의 필수 도구" 반응 |

**배울 점**: AI 에이전트의 "감각 기관"을 만들어주는 보조 도구 시장이 열리고 있다. 에이전트가 못 하는 것(시각적 피드백)을 해결하는 틈새가 곧 기회.

### 2. Claude Code auto-fix ⬆️ 307 | 💬 7
![thumbnail](https://ph-files.imgix.net/fcf35c79-bc8a-4c2b-b23e-705da4d49392.png?auto=format)

📂 Developer Tools, Artificial Intelligence, Development

#### 이 제품은 무엇인가?
클라우드에서 PR을 감시하며 CI 실패와 리뷰 코멘트를 자동으로 해결하는 도구다. PR을 열어두고 자리를 비워도 돌아오면 머지 가능 상태가 되어 있다.
- 핵심 기능: CI 실패 자동 수정 + 리뷰 코멘트 자동 반영 + 필요시 개발자에게 질문

| 구분 | 내용 |

| 기존 대비 차별점 | 코드 생성이 아닌 "PR 유지보수" 특화 — CI/리뷰 자동 수정 |
