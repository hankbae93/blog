**조합 공식**: Export contacts by Covve(iOS TOP PAID 31위, $3.99)의 시장 검증 + 해당 앱의 크래시/기능 부족 불만 (7건 sync/data, 3건 missing features)

**문제 발견 경로**:
- App Store: Export contacts by Covve — 리뷰 1,554개, 평점 4.76, TOP PAID 31위 = 검증된 시장
- App Reviews: 불만 7건 sync/data ("앱이 내보내기 버튼 누르면 크래시"), 3건 missing features ("VCF 형식 지원 안 함"), 2건 pricing ("다른 형식 내보내기에 또 결제")
- HN: Android Developer Verification (74점) — 모바일 개발 생태계에 대한 관심

**문제**: iOS에서 연락처를 CSV, VCF, Excel로 내보내야 하는 사용자(이직, 폰 변경, 백업)가 Export contacts by Covve를 쓰지만, 앱이 크래시하고, VCF 형식을 지원하지 않으며, 형식별로 별도 결제를 요구한다.

**솔루션**: 한 번 구매로 모든 형식(CSV, VCF, Excel, JSON)을 지원하는 안정적인 iOS 연락처 내보내기 앱. 복원 기능도 포함.

**MVP 스펙 (핵심 기능 3개)**:
1. 원터치 내보내기 - CSV, VCF, Excel, JSON 모든 형식 지원, 한 번에
2. 스마트 필터 - 그룹, 회사, 최근 연락 등 기준으로 선택 내보내기
3. 복원 기능 - 내보낸 파일로 연락처 복원 가능

**기술 스택**: Swift + Contacts Framework + 파일 변환 로직 (서버 불필요)

**수익 검증 데이터**:
- 유사 앱 Export contacts by Covve: 리뷰 1,554개 (추정 DAU ~78), TOP PAID 31위, $3.99
- 앱 리뷰 불만: "앱이 크래시한다" 7건, "VCF 지원 안 함" 3건, "형식별 별도 결제" 2건
- 예상 수익 모델: 일회성 $4.99 (경쟁사보다 $1 비싸지만 모든 형식 포함)

- 이 아이디어의 차별점: 크래시 없는 안정성 + 모든 형식 한 번에 + 복원 기능 + 일회성 결제
