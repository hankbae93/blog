### 카드 3: AgentLog — AI 에이전트 활동 모니터링 대시보드

**조합 공식**: Agent Safehouse(HN 233점)의 에이전트 보안 관심 + Pulldog(PH 172표)의 "개발자 워크플로 네이티브 대시보드" 패턴

**문제 발견 경로**:
- Hacker News: Agent Safehouse (233점, 52개 댓글) — 로컬 에이전트 보안에 대한 폭발적 관심
- Hacker News: 문학적 프로그래밍 재고 (97점) — 에이전트가 한 일을 추적·이해하는 문제
- Lobsters: human.json (34점) — AI와 인간 행동의 구분 필요성

**문제**: Claude Code, Cursor, Copilot 같은 AI 코딩 에이전트가 파일을 수정하고 명령을 실행하지만, "에이전트가 정확히 무엇을 했는지" 한눈에 보기 어렵다. git diff로 결과는 보이지만, 어떤 순서로 어떤 판단을 내렸는지는 모른다.

**솔루션**: AI 에이전트의 파일 변경, 명령어 실행, 의사결정 과정을 타임라인으로 시각화하는 웹 대시보드.

**MVP 스펙 (핵심 기능 3개)**:
1. **로그 수집** - .claude/, .cursor/ 등 에이전트 로그 파일 파싱, 파일 변경 이벤트 감지 (fs.watch)
2. **타임라인 뷰** - 시간순으로 에이전트 행동 시각화 (파일 수정, 명령 실행, 도구 호출)
3. **세션 요약** - 한 세션의 변경 파일 목록, 실행 명령어, diff 요약을 한 페이지에 표시

**기술 스택**: Next.js + Node.js (로그 파서) + SQLite (세션 저장)

**수익 검증 데이터**:
- Agent Safehouse: HN 233점으로 에이전트 보안/모니터링 수요 검증
