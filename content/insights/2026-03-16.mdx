2. 단계별 제작 가이드 — 실 색상, 스티치 순서, 필요 재료 자동 산출
3. 패턴 PDF 다운로드 — 인쇄 가능한 고품질 패턴 시트

**기술 스택**: Next.js + OpenAI API (Vision + GPT) + Canvas/SVG 렌더링

**1인 개발자 실현 가능성**: ⭐⭐⭐⭐☆ (4/5)

### 카드 3: AgentAudit — 에이전트 스킬 보안 스캐너 (경량 CLI)

**조합 공식**: ClawSecure(PH 271↑, "41% 스킬이 위험")의 보안 수요 + Google Workspace CLI(319↑)의 "MCP 컨텍스트 세금 없이" 접근 + Glassworm 유니코드 공격(HN 206점)의 공급망 보안 경고

**문제 발견 경로**:
- Product Hunt: ClawSecure 60개 댓글 — 에이전트 보안에 대한 강렬한 수요
- Hacker News: Glassworm 206점, 118댓글 — 코드 공급망 공격 재등장
- YouTube: "코드 작성에서 에이전트 관리로" 20만 뷰 — 에이전트 운영 역량 필요

**문제**: OpenClaw 스킬/에이전트를 설치할 때, 해당 스킬이 안전한지 빠르게 확인할 방법이 없다. ClawSecure는 풀 플랫폼이지만, 개인 개발자에게는 무겁다.

**솔루션**: `npx agent-audit check <skill-name>` 한 줄로 에이전트 스킬의 보안 위험을 스캔하는 경량 CLI 도구. 권한 과다 요청, 데이터 외부 전송, 알려진 취약 패턴을 탐지.

**MVP 스펙 (핵심 기능 3개)**:
1. 스킬 퍼미션 분석 — 요청하는 권한 목록을 파싱하고 과다 요청 경고
2. 네트워크 요청 패턴 감지 — 외부 서버로 데이터를 전송하는 코드 패턴 탐지
3. 알려진 위험 스킬 DB — 커뮤니티 기여 기반 블랙리스트/그레이리스트

**기술 스택**: Node.js CLI (Commander.js) + AST 파서 (Babel) + SQLite (로컬 DB)

**수익 검증 데이터**:
- ClawSecure: 271↑, 60댓글 — 보안 도구 수요 검증
- Glassworm 기사: 206점, 118댓글 — 공급망 보안 관심 폭발
- 예상 수익 모델: CLI 무료 + Pro 대시보드 $9.99/월 (팀 리포트, CI/CD 통합)
