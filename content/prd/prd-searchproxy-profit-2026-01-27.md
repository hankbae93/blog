## 1. 개요

### 1.1 문제 정의

인디해커와 소규모 SaaS 운영자들은 검색 결과 스크래핑이 필요하지만, 기존 솔루션(SerpApi, ScraperAPI)의 높은 비용이 진입 장벽이 되고 있다.

**현재 시장의 문제점:**
- **높은 비용**: SerpApi 월 $75 (5,000 req), ScraperAPI 월 $49 (100,000 req, 기본 기능만)
- **과도한 기능**: 대기업 타겟 기능으로 인해 불필요한 비용 부담
- **유연성 부족**: 사이드 프로젝트나 MVP 단계에 맞지 않는 최소 요금제
- **복잡한 가격 체계**: 숨겨진 비용과 복잡한 크레딧 시스템

**시장 기회:**
- 전 세계 인디해커 커뮤니티 성장 (Indie Hackers, Product Hunt 활성 사용자 증가)
- SaaS 생태계 확장으로 검색 데이터 수요 증가
- 비용 효율적인 대안에 대한 명확한 수요 존재

### 1.2 솔루션

**SearchProxy**는 프록시 풀과 지능형 캐싱 시스템을 활용하여 기존 경쟁사 대비 80% 저렴한 가격에 검색 API를 제공한다.

**핵심 가치 제안:**
1. **가격 경쟁력**: $9/월부터 시작하는 합리적인 가격
2. **단순함**: 복잡한 설정 없이 API 키 하나로 시작
3. **신뢰성**: 스마트 프록시 로테이션으로 높은 성공률 보장
4. **효율성**: 캐싱으로 반복 요청 비용 절감

**차별화 포인트:**
- 인디해커 전용 설계 (불필요한 기능 제거)
- 투명한 가격 정책 (숨겨진 비용 없음)
- 커뮤니티 중심 성장 전략

### 2.4 Use Case Scenarios

**Scenario 1: SEO 도구 개발**
> 민준은 키워드 순위 추적 도구를 만들고 있다. 매일 1,000개 키워드를 체크해야 하는데, SerpApi로는 월 $150 이상 든다. SearchProxy $29 플랜으로 월 25,000회 요청이 가능해 70% 이상 절감할 수 있다.

**Scenario 2: 가격 모니터링 봇**
> 서연은 경쟁사 가격을 모니터링하는 SaaS를 운영한다. 하루 5,000회 검색이 필요한데, SearchProxy의 캐싱으로 실제 요청은 2,000회로 줄어들어 $29 플랜으로 충분하다.

**Scenario 3: 리서치 자동화**
> 마케팅 에이전시에서 클라이언트별 경쟁사 분석을 자동화하려 한다. 월 50,000회 검색이 필요하며, $79 플랜으로 기존 대비 60% 비용 절감이 가능하다.

## 3. 기능 요구사항

### 3.1 MVP 기능 (Phase 1)

#### MVP-1: Google 검색 API
**우선순위**: P0 (필수)

**설명**: 기본 Google 검색 결과를 JSON 형태로 반환

**요구사항:**
- Google SERP 파싱 (제목, URL, 스니펫, 위치)
- 지역/언어 설정 지원 (gl, hl 파라미터)
- 페이지네이션 지원 (start, num 파라미터)
- 모바일/데스크탑 결과 구분

**API 엔드포인트:**
```
GET /v1/search
```

**파라미터:**
| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| q | string | Yes | 검색 쿼리 |
| gl | string | No | 국가 코드 (기본: us) |
| hl | string | No | 언어 코드 (기본: en) |
| num | int | No | 결과 수 (1-100, 기본: 10) |
| start | int | No | 시작 위치 (기본: 0) |
| device | string | No | desktop/mobile (기본: desktop) |

**응답 예시:**
```json
{
  "search_metadata": {
    "id": "search_abc123",
    "status": "success",
    "created_at": "2026-01-27T10:30:00Z",
    "processed_at": "2026-01-27T10:30:01Z",
    "total_time_taken": 1.2
  },
  "search_parameters": {
    "q": "best coffee shops",
    "gl": "us",
    "hl": "en"
  },
  "organic_results": [
    {
      "position": 1,
      "title": "10 Best Coffee Shops in NYC",
      "link": "https://example.com/coffee",
      "snippet": "Discover the best coffee shops...",
      "displayed_link": "example.com"
    }
  ],
  "pagination": {
    "current": 1,
    "next": "/v1/search?q=best+coffee+shops&start=10"
  }
}
```

**성공 기준:**
- 응답 시간 < 3초 (p95)
- 성공률 > 95%
- 정확도 > 98%

#### MVP-3: 스마트 캐싱
**우선순위**: P0 (필수)

**설명**: 동일 검색어에 대한 캐시 응답으로 비용 및 속도 최적화

**요구사항:**
- TTL 기반 캐시 (기본 1시간, 최대 24시간)
- 캐시 키: query + gl + hl + device
- 캐시 히트 시 요청 미차감
- 사용자별 캐시 설정 가능

**캐시 정책:**
| 캐시 타입 | TTL | 사용 사례 |
|-----------|-----|-----------|
| Fresh | 1시간 | 실시간 순위 추적 |
| Standard | 6시간 | 일반 검색 (기본값) |
| Extended | 24시간 | 리서치, 분석 |

**API 파라미터:**
```
GET /v1/search?q=keyword&cache=standard
```

**응답 헤더:**
```
X-Cache-Status: HIT | MISS
X-Cache-TTL: 3600
X-Request-Charged: true | false
```

**성공 기준:**
- 캐시 히트율 > 30%
- 캐시 응답 시간 < 100ms
- 캐시 정확도 100%

### 4.4 보안 요구사항

**인증/인가:**
- API 키 SHA-256 해싱 저장
- Rate Limiting (IP + API 키 기반)
- JWT 토큰 만료: 24시간

**데이터 보호:**
- 전송 중 암호화: TLS 1.3
- 저장 시 암호화: AES-256
- PII 로깅 금지

**인프라 보안:**
- DDoS 방어 (Cloudflare)
- WAF 적용
- 정기 보안 감사 (분기별)

### 8.4 API 코드 예제

**Python:**
```python
import requests

API_KEY = "your_api_key"
BASE_URL = "https://api.searchproxy.io"

response = requests.get(
    f"{BASE_URL}/v1/search",
    headers={"Authorization": f"Bearer {API_KEY}"},
    params={
        "q": "best coffee shops NYC",
        "gl": "us",
        "hl": "en",
        "num": 10
    }
)

data = response.json()
for result in data["organic_results"]:
    print(f"{result['position']}. {result['title']}")
    print(f"   {result['link']}")
```

**JavaScript:**
```javascript
const axios = require('axios');

const API_KEY = 'your_api_key';
const BASE_URL = 'https://api.searchproxy.io';

async function search(query) {
  const response = await axios.get(`${BASE_URL}/v1/search`, {
    headers: { 'Authorization': `Bearer ${API_KEY}` },
    params: { q: query, gl: 'us', hl: 'en', num: 10 }
  });

  return response.data.organic_results;
}

search('best coffee shops NYC').then(results => {
  results.forEach(r => console.log(`${r.position}. ${r.title}`));
});
```
