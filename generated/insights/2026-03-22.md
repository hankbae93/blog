#### 무엇을 의미하는가?

<Callout type="info">
AI 에이전트가 "코드 생성"이라는 단일 기능에서 "디자인 + 워크플로우 관리 + 플랫폼 앱 빌딩"까지 확장되고 있다. B2A(Business-to-Agent) 시장이 형성 중.
</Callout>

PH 상위 5개 중 4개가 에이전트 도구라는 것은 단순 트렌드가 아니라 시장 구조의 변화. 에이전트가 "도구"에서 "사용자"가 되면서, 에이전트를 위한 도구·서비스·API가 새로운 카테고리가 되고 있다.

#### 기회 또는 경고

<Steps>
### 아이디어 1: 에이전트 품질 검증 도구

### 아이디어 2: 에이전트 비용 모니터링
여러 에이전트를 동시에 운영할 때 API 호출 비용, 토큰 사용량을 실시간으로 추적하고 최적화 제안을 해주는 대시보드.
</Steps>

</Callout>

Microsoft조차 과도한 AI 기능을 롤백한 것이 상징적. Comet 브라우저의 "Google 검색할 때마다 Perplexity가 끼어든다"는 불만도 같은 맥락. 사용자는 AI가 도움이 되길 원하지, 모든 곳에 AI가 끼어드는 것을 원하지 않는다.

#### 기회 또는 경고

<Steps>

"이 PR의 코드 중 AI 생성 부분"을 식별하고, 해당 부분의 테스트 커버리지/보안 취약점을 집중 검사하는 도구.
</Steps>
